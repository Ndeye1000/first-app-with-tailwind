# First App with Tailwind

Une application web simple créée avec Vite et Tailwind CSS.

## 🚀 Installation

```bash
npm install
```

## 🛠️ Développement

Pour démarrer le serveur de développement :

```bash
npm run dev
```

## 📦 Production

Pour construire l'application pour la production (création du dossier `dist` optimisé) :

```bash
npm run build
```

Pour tester localement la version de production :

```bash
npm run preview
```

Ces commandes sont essentielles avant de déployer votre application sur un serveur de production.

## 🔧 Technologies Utilisées

- [Vite](https://vitejs.dev/) - Outil de build ultra-rapide
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitaire
- JavaScript vanilla
